Photo Name Search

Use this page to search the Kinghorn Photos with Names album. Enter the first three letters or more of a surname (for example, McA for McAndrew) to see the photo file(s) they appear in.

Tick entries in the results. First choose a Copy mode noting that Photo only allows you to select just one photo file that you can Copy then Open the Album and paste into it's search box to find and display the image. Selecting Name + photo allows you to copy and save multiple selected entries to your device.
