# Version 0.5.1 — Framework Freeze

Version 0.5.1 closes the template-design phase and establishes the stable working environment for the Volume I manuscript.

## Frozen components

- Eight master pages
- A4 facing-page geometry and margins
- Typography and paragraph/character style set
- Running headers and automatic page-number frames
- Standard Feature Record and evidence-analysis structures

## Deliberate pagination behaviour

The title page, copyright page and intentionally blank page use masters without a visible page number. All other generated working pages use numbered masters.

## Defect corrections

- Removed document design guides from production output.
- Repositioned and reduced the Chapter Status panel to avoid the footer.
- Replaced demonstration/specimen content with named manuscript working pages.
- Added generator, manuscript, generation date, status and repository metadata.

After local validation, changes to the framework should be limited to confirmed defects. Manuscript content, evidence and figures become the principal development work.
